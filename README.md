# Pokemon GOGO
Pokemon GoGo is a small cli app interacting with the [PokeApi](https://pokeapi.co) made in order to discover go and it's standard library while my wife was playing Pokemon Z-A. 

## Quick Start

```sh
go install github.com/the-1aw/pokemon-gogo
```
```sh
pokemon-gogo
```

## Features
- List location areas
- Show pokemon encountered in an area
- Try to catch pokemon
- Inspect pokemon you've caught
- List all the pokemon you've caught 

## Usage

```
map: Display the next 20 locations
mapb: Display the previous 20 locations
pokedex: displays all the pokemon you caught
catch: <pokemon-name> throw a pokeball at <pokemon-name>
exit: Exit the Pokedex
explore: <location-area> display pokemon encounted in the location-area
help: Displays a help message
inspect: <pokemon-name> display info of captured pokemon
```
